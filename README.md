### Getting Started
#### First step:
```
clone repository
$ git clone ...
$ cd company-app
```

#### Second step:
In the project directory, you can run:

```
$ docker-compose build
```

#### Third step:
```
$ docker-compose up
```

Runs the app.<br>

Open [http://localhost/login](http://localhost/login) to view it in the browser.


```
# kill containers (DB data will be lost)
$ docker-compose down
```
